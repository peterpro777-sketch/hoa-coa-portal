Secure HOA/COA portal for owners and office staff: notices, minutes, agendas, elections and voting.

== Installation ==
1. Upload the ZIP via Plugins → Add New → Upload Plugin.
2. Activate HOA/COA Portal.
3. Create a page and add shortcode: [hcp_portal]
4. Assign users the role "HCP Owner" or "HCP Office Staff".
